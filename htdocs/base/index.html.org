<p>Congratulations!  CMap has been installed.</p>
<p>Eventually you will want to create your own content for this intro page.</p>
<p>Start using CMap with one of the following options.</p>
<ul><li><a href="/cgi-bin/cmap/viewer">Maps</a> - Use a menu to select your starting maps
</li> <li><a href="/cgi-bin/cmap/map_search">Map Search</a> - If the map set is quite large, the Map Search page can be quicker than sorting through menus.
</li> <li><a href="/cgi-bin/cmap/feature_search">Feature Search</a> - Search for a specific feature and display it on a map.
</li> <li><a href="/cgi-bin/cmap/matrix">Matrix</a> - View a table of the number of correspondences between pairs of map sets and maps.
</li> <li><a href="/cgi-bin/cmap/saved_link">Saved Links</a> - View pages previously saved or imported.
</li></ul>
<p>For an introduction to the basic consepts of CMap, please see the <a href="/cgi-bin/cmap/help">help pages</a> or the <a href="/base/tutorial">tutorial</a>.
